# My Personal Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, and professional background.

## 🚀 Features

- **Responsive Design**: Optimized for all devices, including desktops, tablets, and phones.
- **Dynamic Content**: Interactive elements and smooth animations using `particles.js`.
- **Project Showcase**: A dedicated section to highlight my latest work and achievements.
- **Contact Form**: Integrated contact section for professional inquiries.

## 🛠️ Tech Stack

- **HTML5**: For structured and semantic web content.
- **CSS3**: For modern styling, including glassmorphism and custom animations.
- **JavaScript (ES6+)**: For interactive logic and dynamic features.
- **Particles.js**: For the animated background effects.

## 📂 Project Structure

```text
├── index.html       # Main entry point
├── styles.css       # Custom styling and animations
├── main.js         # Core interactive logic
├── particles.js     # Background animation library
└── assets/          # Images and documents (profile photo, CV, etc.)
```
