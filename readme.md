# 💻 Personal Portfolio

Welcome to my professional personal portfolio! This project serves as a centralized hub to showcase my technical skills, academic background, and the various web development projects I've built.

## 🚀 Key Features

* **Interactive UI:** Features a dynamic particle background using `particles.js` for an engaging user experience.
* **Fully Responsive:** Designed with a mobile-first approach to ensure a seamless experience across all devices.
* **Modern Aesthetics:** Uses glassmorphism and custom CSS animations for a sleek, professional look.
* **Project Gallery:** Highlights my work with direct links to live demos and source code.
* **Downloadable Resume:** Integrated link to my current CV for professional inquiries.

## 🛠️ Built With

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Libraries:** Particles.js
* **Icons & Assets:** Custom cursors and high-quality imagery
